Longest Substring Without Repeating Characters (3 medium)

Given a string, find the length of the longest substring without repeating characters. For example for string “pwwkew”, the longest substring is “wke”.

Solution 1. Variable to keep the max value 2. Two pointer (one to iterate the string, and another to keep the beginning of the substring)

Longest common prefix (14 easy)

Example: “abcde”, “abcz”, “abcxy”, “abcd”

Solution 1. Get the prefix of the first string 2. Make use of the indexOf method.

Longest Palindromic Substring (5 medium)

Given a string s, find the longest palindromic substring in s. Example: 1. Input: “babad”, Output: “bab”, Note: “aba” is also a valid answer. 2. Input: “cbbd”, Output: “bb”.

Solution: 1. Consider two situation: oddsituation(“aba”) and evensituation(“bb”) 2. Iterate over the whole string. For each character at index i, check whether the two situation satisfied. If satisfied, return the palindromic substring.
